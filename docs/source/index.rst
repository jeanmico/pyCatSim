PyCatSim
=========

PyCatSim is a toy Python package that simulates cats and their associated humans and environment. The package is meant to teach packaging techniques and proper documentation.

PyCatSim has been made freely available under the terms of the `Apache 2.0 License <https://github.com/khider/pyCatSim/blob/main/LICENSE>`_. There are many ways to get involved :ref:`with the development of PyCatSim<contributing_to_pycatsim>`:

* If you write a paper making use of PyCatSim, please cite it :ref:`thus <citing_pycatsim>`.
* Report bugs and problems with the code or documentation to our `GitHub repository <https://github.com/khider/pyCatSim/issues>`_. Please make sure that there is not outstanding issues that cover the problem you're experiencing.
  * Contribute bug fixes
  * Contribute enhancements and new features
  * Contribute to the code documentation
